# 아쉬탕가 시퀀스 게임

외부 공개용으로 분리한 Next.js 기반 아쉬탕가 프라이머리 시퀀스 게임입니다.

## 포함 기능

- 프라이머리 시퀀스 게임
- Primary Level 1-7
- 영어 시퀀스 게임 MVP
- 선택지 방식 퀴즈
- 점수, 콤보, 라이프, 타이머
- 로그인 없음
- DB 없음
- Prisma 없음
- NextAuth 없음

## 실행

```bash
pnpm install
pnpm dev
```

브라우저에서 `http://localhost:3000`으로 접속합니다.

## 빌드

```bash
pnpm build
```

## 배포

GitHub에 이 폴더를 독립 저장소로 올린 뒤 Vercel에서 Next.js 프로젝트로 import하면 됩니다.
